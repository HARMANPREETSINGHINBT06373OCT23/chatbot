About Us

Our chatbot is built using Python Flask along with several useful libraries to manage user interaction efficiently. The application is designed to provide an interactive experience with users by answering their questions or providing relevant responses.

The chatbot uses a database (stored in database.py) which contains a list of frequently asked questions and answers. This ensures that the bot is equipped with relevant information to assist users.

Flask is a lightweight Python web framework that helps us to handle requests, manage user inputs, and interact with the database seamlessly.
